# Solar Powered WiFi Weather Station V3.0

A fully autonomous **solar powered IoT weather station** built using the **ESP32 WiFi module** and multiple environmental sensors.  
The system measures key weather parameters and uploads the data to cloud platforms for **real-time monitoring and data logging**.

The station operates entirely on **solar power with a rechargeable Li-ion battery**, making it suitable for remote installations where grid electricity is not available.

## Measured Weather Parameters

This weather station measures the following environmental parameters:

• Air Temperature  
• Humidity  
• Barometric Pressure  
• External Temperature  
• Wind Speed  
• Wind Direction  
• Rainfall  
• UV Index  
• Light Intensity (Lux)

The system integrates several sensors including **BME280, DS18B20, BH1750, SI1145, and SparkFun Weather Meter sensors** to collect detailed meteorological data.
